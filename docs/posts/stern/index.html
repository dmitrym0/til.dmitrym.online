November 13, 2023 One minute read

stern is a log tailer for k8s. Similar to tail -f but for multipe Kubernetes pods:

1
2
3

stern --only-log-lines  \
      --max-log-requests 1000 \
      --namespace k8s-namesapce \
      'pod-wildcard*'
